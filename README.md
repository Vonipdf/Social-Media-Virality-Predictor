# Social-Media-Virality-Predictor
## Quantitative Visual Feature Analysis for Social Media Virality Prediction

In modern digital media, predicting content engagement ("virality") is a high-value task for brand managers, content creators, and algorithmic recommendation systems. Traditional social media analytics heavily favor text-based metadata (such as captions, hashtags, and posting times). However, contemporary platforms like Instagram and TikTok are visual-first mediums.

This project investigates whether an image's inherent visual characteristics—specifically color vibrancy, structural complexity, and brightness contrast—statistically correlate with its popularity. Using a streaming pipeline of the AVA (Aesthetic Visual Analysis) Dataset, we extract mathematical features directly from pixel arrays and construct a Logistic Regression framework to predict the probability of an image trending based solely on raw visual aesthetics.
